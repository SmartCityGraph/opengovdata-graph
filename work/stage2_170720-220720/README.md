### Второй этап создания онтологии

На данном этапе нам необходимо создать индивиды ОКТМО для онтологии основываясь на предлагаемом файле. Для это мы использовали файл
OKTMO_database.csv - база данных ОКТМО в формате строк. Одна строка - один код ОКТМО.

На основе этого файла мы сгенерировали два вспомогательных файла:
OKTMO_locality_type_base.csv - вручную заполненный файл типов сокращений и их рассшифровок
OKTMO_capital_database.csv - автоматически сгенерированный файл кодов всех столиц ОКТМО

С помощью этих файлов и написанного скрипта мы генерируем большинство индивидов ОКТМО, которые, однако нуждаются в некоторой ручной обработке (в частности индивиды регионов).
OKTMO_ontology_object_database.ttl - индивиды ОКТМО в формате ttl
