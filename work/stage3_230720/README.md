### Третий этап - редактирование базы индивидов и соединение с онтологией

На этом этапе нам необходимо внести небольшие изменения в нашу сгенерированную базу данных, эти изменения отражены в файле
changes.ttl

Список изменений:
- Создание индивидов федеральных округов
- Изменение классов для регионов в соответсвии с их статусом (область, республика и т.д.)
- Создание необходимых связей между регионами и федеральными округами
- Изменение имён регионов для лучшей читаемости

Объединив изменённую базу данных индивидов с онтологией мы получаем рабочий файл
ontology_with_inividuals.ttl
